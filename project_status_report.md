## 🚀 部署建议

### 本地使用
1. 直接双击 `index.html` 文件即可在浏览器中打开使用
2. 或者使用 `demo.html` 进行功能演示和测试
3. 数据会自动保存在浏览器本地存储中

### 服务器部署
由于这是一个纯前端项目，可以部署到任何静态文件服务器：
- GitHub Pages
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS

### 注意事项
- 数据存储在浏览器本地，清除浏览器数据会丢失所有信息
- 建议定期导出重要数据
- 不同浏览器之间的数据不共享
